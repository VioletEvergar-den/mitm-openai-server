OpenAI API 配置指南

什么是 OpenAI API?

OpenAI API 是一组由 OpenAI 公司提供的强大AI模型接口，包括 GPT-3.5、GPT-4 等模型。这些接口允许开发人员将先进的AI能力集成到他们的应用程序中，用于文本生成、摘要、对话、翻译等多种任务。

如何配置您的客户端使用这个服务

要在您的 OpenAI API 客户端中使用这个服务，请按照以下步骤操作：

  1. 在您的 OpenAI 工具中，配置服务器 URL：
  2. 如果启用了 API 认证，您需要配置认证信息
  3. 导入 OpenAI API 规范或使用 URL 直接访问

客户端示例

curl 示例

curl -X POST "http://localhost:8080/v1/chat/completions" \ -H "Content-Type: application/json" \ -d '{ "model": "gpt-3.5-turbo", "messages": [ {"role": "user", "content": "你好，请介绍一下自己"} ] }'

JavaScript 客户端示例

fetch('http://localhost:8080/v1/chat/completions', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ model: 'gpt-3.5-turbo', messages: [ {role: 'user', content: '你好，请介绍一下自己'} ] }) }) .then(response => response.json()) .then(data => console.log(data));

Python 客户端示例

import requests response = requests.post( 'http://localhost:8080/v1/chat/completions', json={ 'model': 'gpt-3.5-turbo', 'messages': [ {'role': 'user', 'content': '你好，请介绍一下自己'} ] } ) print(response.json())
